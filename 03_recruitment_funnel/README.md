# Recruitment Funnel Analysis

## Business Question
HR wants to analyze the recruitment pipeline to identify drop-offs and improve hiring efficiency.

## Objectives
- Track candidates across hiring stages
- Analyze department-wise hiring
- Calculate offer-to-join ratio
- Identify stages with maximum drop-offs

## Dataset
Recruitment data including application date, current status, department, and final hiring decision.

## SQL Skills Used
- SELECT
- COUNT
- CASE WHEN
- GROUP BY
- WHERE

## Key Insights
- Maximum candidate drop-off occurs before interviews
- Certain departments have better conversion rates
- Offer acceptance rate varies by department
