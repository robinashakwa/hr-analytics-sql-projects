# Employee Attrition Analysis

## Business Question
HR wants to understand employee attrition patterns and identify factors contributing to employee exits.

## Objectives
- Calculate overall attrition rate
- Identify department-wise attrition
- Analyze attrition by gender and salary
- Understand tenure-based attrition trends

## Dataset
Employee master data including joining date, exit date, department, salary, gender, and employment status.

## SQL Skills Used
- SELECT
- COUNT
- CASE WHEN
- GROUP BY
- WHERE
- DATE functions

## Key Insights
- Attrition is highest in specific departments
- Employees with lower tenure show higher exit rates
- Salary bands influence attrition trends
