# Workforce Diversity & Inclusion Analysis

## Business Question
HR leadership wants to assess workforce diversity and inclusion to ensure equitable representation across departments and pay levels.

## Objectives
- Analyze gender diversity across the organization
- Identify department-wise diversity distribution
- Evaluate diversity across salary bands
- Support inclusion-focused HR decision making

## Dataset
Employee demographic data including gender, department, salary, and employment status.

## SQL Skills Used
- SELECT
- COUNT
- GROUP BY
- CASE WHEN
- ORDER BY

## Key Insights
- Diversity representation varies across departments
- Certain salary bands show lower diversity
- Data-driven insights can support inclusive HR policies
