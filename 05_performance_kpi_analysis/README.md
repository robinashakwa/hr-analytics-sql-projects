# Performance & HR KPI Analysis

## Business Question
HR leadership wants to evaluate employee performance and track key HR KPIs to support decision-making.

## Objectives
- Analyze performance rating distribution
- Identify department-wise performance trends
- Understand relationship between performance and attrition
- Prepare KPI metrics for HR dashboards

## Dataset
Employee performance data including department, performance rating, salary, and employment status.

## SQL Skills Used
- SELECT
- AVG
- COUNT
- GROUP BY
- CASE WHEN

## Key Insights
- Performance varies significantly across departments
- Lower performance ratings show higher attrition risk
- KPIs help HR monitor workforce health proactively
